Latitude vs. Temperature in the Northern Hemisphere

The purpose of this project was to analyze how weather changes as one gets closer to the equator. For this nalysis, I pulled data from the OpenWeatherMap API and assembled a dataset of over 500 cities.

After assembling and tidying the dataset, I used Matplotlib to plot data and isolate variables, studying changes in weather vs. latitude. This site provides exmples of the variables wind and temperature for the northern and southern hemispheres.
